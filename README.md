# 🛸 UZDF — Платформа для пилотов БПЛА в Узбекистане

**UZDF** — это многофункциональная единая экосистема для пилотов гражданских беспилотных летательных аппаратов (БПЛА) в Республике Узбекистан. Проект включает в себя интерактивную карту воздушного пространства, обучающую академию с выдачей именных цифровых сертификатов, магазин FPV-оборудования и панель администрирования.

---

## 🏗️ Архитектура и Стек технологий

Проект разделен на три ключевые части:

### 1. 🔌 Бэкенд (API-сервер)
* **Технологии**: Node.js, Express, Prisma ORM, PostgreSQL (поддержка SQLite для локальной разработки).
* **Безопасность**: шифрование паролей через `bcrypt`, авторизация на основе `JWT-токенов`.
* **Функционал**:
  * Подтверждение регистрации по электронной почте (6-значный OTP-код).
  * Ролевая модель доступа: `user` (пилот), `moderator`, `admin`, `superadmin`.
  * API для логирования нарушений (например, частые скриншоты в учебных курсах) с автоматической блокировкой на 24 часа.
  * Учет игрового опыта (EXP) и уровней пилота за активность (прохождение курсов, тестов, заказы, считывание погоды).

### 2. 📱 Мобильное приложение (Flutter)
* **Технологии**: Flutter, Dart, `url_launcher`, `shared_preferences`, `http` для сетевых запросов.
* **Функционал**:
  * Автоматическое определение курса на главном промо-баннере ("Продолжить обучение" вместо "Начать" с перенаправлением на самый прогрессирующий курс).
  * Интерактивная карта с отображением полетных зон.
  * Выдвижная панель погоды (`DraggableScrollableSheet`), отображающая статус пригодности ветра, температуры, давления и UV-индекса для полетов.
  * Профиль пилота с игровым уровнем, шкалой опыта (EXP) и списком разблокированных ачивок.
  * Раздел заказов с возможностью указания адреса доставки и ее отслеживанием.
  * Техподдержка с интеграцией Telegram-бота по кнопке.

### 3. 🌐 Веб-версия и Админ-панель
* **Технологии**: HTML5, Vanilla CSS3 (современный неоновый FPV-стиль с темной темой), JavaScript, Leaflet API (карты зон).
* **Академия и Строгое обучение**:
  * Фиксация старта и прохождения уроков на сервере.
  * Отслеживание скроллинга страницы и минимального времени чтения для текстовых лекций (кнопка "Далее" блокируется до выполнения условий).
  * Посекундный таймер просмотра для видеолекций.
  * Безопасные тесты: вопросы загружаются в случайном порядке без правильных ответов. Ответы отправляются разом на сервер (`/quiz-submit`), исключая читерство на клиенте.
* **Админ-панель (`/admin`)**:
  * Мониторинг статистики платформы (выручка, пользователи, курсы, зоны).
  * Создание новых администраторов и суперадминистраторов (только для роли `superadmin`).
  * Изменение ролей пользователей через выпадающие списки в таблице.
  * Детальный просмотр прогресса пользователей, сброс лимита попыток тестов, сброс таймеров видео, история активности и лог нарушений.
  * Интерактивный Leaflet-редактор: рисование геозон (RED, YELLOW, GREEN) прямо на карте мышкой с сохранением координат в БД.

## 🚀 Как запустить проект

### 1. Настройка Бэкенда
1. Перейдите в папку бэкенда:
   ```bash
   cd backend
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Создайте файл `.env` на основе примера и укажите параметры подключения к базе данных.
4. Примените миграции базы данных и запустите сидинг:
   ```bash
   npx prisma db push
   node seed.js
   node seed_about.js
   node seed_zones.js
   node seed_advanced_courses.js
   ```
5. Запустите сервер:
   ```bash
   node server.js
   ```
   Сервер запустится на порту `3000` (публичный сайт доступен по адресу `http://localhost:3000`, админка по `http://localhost:3000/admin`).

### 2. Запуск Мобильного приложения (Flutter)
1. Убедитесь, что у вас установлен Flutter SDK.
2. Перейдите в каталог приложения:
   ```bash
   cd mobile
   ```
3. Получите зависимости:
   ```bash
   flutter pub get
   ```
4. Запустите приложение на симуляторе или реальном устройстве:
   ```bash
   flutter run
   ```
